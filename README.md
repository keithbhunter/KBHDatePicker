# KBHDatePicker
Infinite scrolling date picker similar to iOS' Calendar app.

## Usage

```
let frame = CGRectMake(0, 44, self.view.bounds.width, 70)
let datePicker = KBHDatePicker(frame: frame)
self.view.addSubview(datePicker)
```
